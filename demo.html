Abbreviations

Abbreviations make use of simulated small caps, as in HTML, because most fonts do not implement a suitable size of smallcaps for use in place of uppercase letters in running text (they are implemented as x-height petite caps instead, which are more suited for replacing lowercase letters). The font size and weight of such <abbr> elements need to be tuned in the stylesheet to properly fit the font in use. CSS is fun; for abbreviations that start a sentence, the class .g-capital should be used to ensure that the first letter is fully capitalised.
